# Zoom Image Generator

Zoom 회의 화면처럼 보이는 2x2 참가자 이미지를 만들고 PNG로 저장할 수 있는 웹 기반 도구입니다.

## 주요 기능

- Zoom 스타일 상단 바와 하단 툴바가 포함된 회의 화면 생성
- 참가자 4명의 사진, 닉네임, 위치를 각각 설정
- 사진 맞춤 방식 선택: 화면 채우기, 전체 보이기
- 참가자별 확대, 좌우, 상하 위치 조정
- 참가자별 배경 블라인드 처리 켜기/끄기
- 참가자별 초록색 테두리 표시 켜기/끄기
- 화면 구분선과 상단/하단 바 표시 옵션
- 완성된 화면을 PNG 이미지로 저장

## 실행 방법

Node.js가 설치되어 있다면 아래 명령어로 실행할 수 있습니다.

```bash
node server.js
```

실행 후 브라우저에서 아래 주소로 접속합니다.

```text
http://127.0.0.1:8787/
```

## 사용 방법

1. 참가자별로 사진을 선택합니다.
2. 닉네임과 화면 위치를 설정합니다.
3. 필요하면 확대, 좌우, 상하 값을 조정합니다.
4. 배경 블라인드 또는 초록 테두리 옵션을 선택합니다.
5. 미리보기를 확인한 뒤 `PNG 저장` 버튼으로 이미지를 저장합니다.

## 참고

이 프로젝트는 실제 Zoom 화면을 그대로 사용하는 프로그램이 아니라, Zoom 회의 화면과 비슷한 형식의 PNG 이미지를 만들기 위한 개인용 이미지 생성 도구입니다.
